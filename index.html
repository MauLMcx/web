Tentang Saya

Pelajar SMK jurusan Teknik Jaringan Komputer dan Telekomunikasi dengan passion di jaringan dan pengembangan software

Halo, Saya Maula Herga Nusantara

Saya adalah pelajar SMK jurusan Teknik Jaringan Komputer dan Telekomunikasi (TJKT) yang memiliki minat besar pada dunia jaringan, sistem operasi berbasis Linux, dan pengembangan mod untuk Minecraft.

Saya aktif belajar konfigurasi jaringan (OSPF, EIGRP, RIP), virtualisasi menggunakan VirtualBox, serta eksplorasi server berbasis Debian dan Arch Linux. Di luar jaringan, saya memiliki passion dalam pengembangan mod Minecraft client-side menggunakan Fabric API, Java, dan JSON.
